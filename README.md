## 📋 Overview

**TenderScan.ai** is an intelligent procurement solution that streamlines the analysis of complex tender documents. By leveraging advanced AI capabilities, it automatically identifies, extracts, and categorizes mandatory requirements from tender documentation, saving time and reducing manual effort in the procurement process.

### Key Benefits
- ⚡ **Fast Analysis** - Process complex tender documents in seconds
- 🎯 **Accurate Extraction** - Identify mandatory requirements with high precision
- 📊 **Smart Categorization** - Automatically organize requirements by type
- 🚀 **Easy to Use** - Intuitive interface for seamless user experience
- 🤖 **AI-Powered** - Built with cutting-edge AI technology

## 📦 Installation

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn package manager

### Setup Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/DhruvPatil123/TenderScan.Ai.git
   cd TenderScan.Ai
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   - Open `.env.local` file
   - Add your Gemini API key:
     ```
     GEMINI_API_KEY=your_api_key_here
     ```
   - Get your API key from [Google AI Studio](https://ai.google.dev/)

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. **Open in browser**
   - Navigate to `http://localhost:3000`
   - Start analyzing tender documents!
